Data Quality Notes

  • Sentiment analysis uses VADER (Valence Aware Dictionary and sEntiment Reasoner) which is optimized for social media text and survey responses
  • VADER compound scores: ≥ 0.05 = positive, ≤ -0.05 = negative, between = neutral
  • Program awareness is based on case-insensitive keyword matching in text responses
  • Barrier analysis uses keyword clustering to identify themes in open-ended responses
  • Some metrics (applicant journey, district-level) are approximations based on available data patterns
  • All percentages are rounded to one decimal place for readability
